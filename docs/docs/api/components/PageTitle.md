# PageTitle

Component used to set the `<title>` tag on the page.

## Props

* `title = {string}`

## Example with code

```jsx
// highlight-start
import PageTitle from '@components/PageTitle';
// highlight-end

const MyComponent = () => {
  return (
    // highlight-start
    <PageTitle title="My page" />
    // highlight-end
  );
};
```
