# Title

`<h1>` tag with pre-applied styles.

## Props

* `children = {React.ReactNode}`
  * React children prop

## Example with code

```jsx
// highlight-start
import Title from '@components/Title';
// highlight-end

const MyComponent = () => {
  return (
    // highlight-start
    <Title>My title</Title>
    // highlight-end
  );
};
```
